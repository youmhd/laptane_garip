CE QU'IL FAUT SAVOIR A PROPOS DE L'APPLICATION

Signification du nom du projet :


Laptane : Hôpital
Garip : Le plus proche

Laptane Garip est une application permettant de localisé les
structures sanitaires le plus proche. En effet, le but de la
conception de cette application est de recencé tout les hôpitaux,
les cliniques, et les centres de santé de la ville de Ndjamena
afin d'avoir avec preecision le local sanitaire environnant. cette
application est conçue pour les cas des urgences ( les accidents de la route,
les maladies chroniques tel le paludisme etc.)
